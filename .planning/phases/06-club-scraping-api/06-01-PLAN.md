    1. Create src/scraper/clubs.ts with two exported functions:

    **scrapeClubList():**
    ```typescript
    import axios from 'axios';
    import * as cheerio from 'cheerio';
    import type { ClubInfo } from '../../packages/shared/types.js';

    const SEARCH_URL = 'https://www.chess.org.il/Players/SearchPlayers.aspx';
    const USER_AGENT = 'ChessIL-Dashboard/1.0 (community project)';
    const AXIOS_CONFIG = {
      headers: { 'User-Agent': USER_AGENT },
      timeout: 15000,
      responseType: 'text' as const,
      validateStatus: () => true,
    };
    ```

    - Step 1: `axios.get(SEARCH_URL, AXIOS_CONFIG)` -- extract `#__VIEWSTATE`, `#__EVENTVALIDATION`, `#__VIEWSTATEGENERATOR` via cheerio
    - Step 2: `axios.post(SEARCH_URL, form2.toString(), { ...AXIOS_CONFIG, headers: { ...AXIOS_CONFIG.headers, 'Content-Type': 'application/x-www-form-urlencoded' } })`
      - form2 fields: `__EVENTTARGET` = `'ctl00$ContentPlaceHolder1$AdvancedSearchLinkButton'`, `__EVENTARGUMENT` = `''`, `__VIEWSTATE` = vs1, `__EVENTVALIDATION` = ev1, `__VIEWSTATEGENERATOR` = vsg1, `ctl00$ContentPlaceHolder1$SearchNameBox` = `''`
    - Parse response: `$2('#ctl00_ContentPlaceHolder1_ClubsDDL option').each(...)`, skip options with `value="0"` (parseInt value, skip if <= 0), push `{ id, name: $(el).text().trim() }`
    - Return type: `Promise<ClubInfo[]>`
    - On any error: log to console.error and return `[]`

    **searchClubPlayers(clubId: number, minAge?: number, maxAge?: number):**
    - Step 1: Same as scrapeClubList step 1

        - `__VIEWSTATE` = vs2 (from step 2 response)
        - `__EVENTVALIDATION` = ev2 (from step 2 response)
        - `__VIEWSTATEGENERATOR` = vsg2 (from step 2 response)
        - `__EVENTTARGET` = `''`
        - `__EVENTARGUMENT` = `''`
        - `ctl00$ContentPlaceHolder1$AdvancedSearchNameTextBox` = `''`
        - `ctl00$ContentPlaceHolder1$ClubsDDL` = `String(clubId)`
        - `ctl00$ContentPlaceHolder1$AgeFromTB` = minAge !== undefined ? `String(minAge)` : `''`
        - `ctl00$ContentPlaceHolder1$AgeTillTB` = maxAge !== undefined ? `String(maxAge)` : `''`
        - `ctl00$ContentPlaceHolder1$GenderDDL` = `'ALL'`
        - `ctl00$ContentPlaceHolder1$CitiesDDL` = `'All'` (NOTE: capital A, lowercase ll -- different from other fields)
        - `ctl00$ContentPlaceHolder1$RatingFromTB` = `''`
        - `ctl00$ContentPlaceHolder1$RatingUptoTB` = `''`
        - `ctl00$ContentPlaceHolder1$ForeignDDL` = `'ALL'`
        - `ctl00$ContentPlaceHolder1$CountriesDDL` = `'ALL'`
        - `ctl00$ContentPlaceHolder1$MembershipStatusDDL` = `'ALL'`
        - `ctl00$ContentPlaceHolder1$PlayerStatusDDL` = `'1'` (active players)
        - `ctl00$ContentPlaceHolder1$AdvancedSearchButton` = `'\u05D7\u05D9\u05E4\u05D5\u05E9'` (Hebrew "חיפוש")
        - DO NOT include `FreeOnlyCB` or `ManagersOnlyCB` (unchecked checkboxes must be omitted)
    - Parse results from `$3('#ctl00_ContentPlaceHolder1_playersGridView')`:
      - Skip header row (index 0)
      - For each data row, extract td cells:
        - Name: `cells.eq(1).find('a').text().trim()` -- fallback to `cells.eq(1).text().trim()`
        - ID: from href match `/Id=(\d+)/` on column 1 link, fallback to `parseInt(cells.eq(2).text().trim(), 10)`
        - Rating: `parseInt(cells.eq(9).text().trim(), 10)` -- if NaN, set null
        - Club: `cells.eq(6).text().replace(/\s+/g, ' ').trim()`
        - Birth year: `parseInt(cells.eq(13).text().trim(), 10)` -- if NaN, set null
      - Skip rows with no valid ID
    - Return type: `Promise<ClubSearchResult[]>`
    - On any error: log to console.error and return `[]`
