## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px (`gap-1`, `p-1`) | Icon gaps, inline chip padding, W/D/L chip internal spacing |
| sm | 8px (`gap-2`, `p-2`, `py-2`) | Icon button padding, compact element spacing, badge padding |
| md | 16px (`gap-4`, `p-4`, `px-4`) | Card internal padding, default element spacing, section horizontal padding |
| lg | 24px (`gap-6`, `py-6`) | Spacing between dashboard sections, card group gaps |
| xl | 32px (`gap-8`, `py-8`) | Page top/bottom padding, major section breaks |
| 2xl | 48px (`py-12`) | Reserved (not used in this phase) |
| 3xl | 64px (`py-16`) | Reserved (not used in this phase) |

Exceptions: Chart container uses `height: 300px` (line/area/bar chart) and `height: 250px` (donut chart) -- these are Recharts ResponsiveContainer dimensions, not spacing tokens.

**Source:** Established codebase patterns from Phase 2 (`px-4`, `py-8`, `gap-2`, `gap-3`, `p-4`). Scale confirmed as 4px base.

## Color

| Dominant (60%) | `#F9FAFB` (`bg-gray-50`) / `#111827` (`dark:bg-gray-900`) | Page background surface |

### 3. Rating Chart (DASH-03, DASH-04, D-05, D-06, D-07, D-08)

| Property | Value |
|----------|-------|

## Responsive Layout Contract

| Breakpoint | Layout |
|------------|--------|
| < 768px (mobile) | Single column. All sections stack vertically: Header -> Metrics (2x2) -> Chart (full-width) -> Donut (full-width) -> Tournament cards (stacked). |

## Accessibility Contract

| Requirement | Implementation |
|-------------|----------------|
| RTL direction | `dir="rtl"` on root -- inherited by all components (established Phase 2) |
| Logical CSS properties | `ms-`/`me-`/`ps-`/`pe-` only -- no physical `ml-`/`mr-`/`pl-`/`pr-` |
| Icon-only buttons | All icon buttons have `aria-label` in Hebrew |
| Chart accessibility | Recharts renders SVG with implicit structure; tooltip provides data on hover |
| External links | `target="_blank" rel="noopener noreferrer"` on all `chess.org.il` and FIDE links |
| Color not sole indicator | Rating changes use both color AND arrow icon direction |
| Pagination buttons | `disabled` attribute when at first/last page |
